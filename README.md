# Morgan Gil

Static artist portfolio for GitHub Pages.

## Structure

This site uses plain HTML, CSS, and JavaScript. There is no build step.

Pages:

- index.html
- work.html
- biography.html
- exhibitions.html
- contact.html
- field-studies.html
- built-fragments.html
- edge-conditions.html
- anatomy-of-the-network.html

## GitHub Pages

Set Pages source to the main branch and root.

The custom domain file is already set to:

```text
morgangil.com
```

At the domain registrar, point the apex domain to GitHub Pages using A records:

```text
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

For www, use a CNAME record pointing to:

```text
anthem-research.github.io
```

Then set the custom domain in GitHub Pages to morgangil.com and enable HTTPS after DNS resolves.
